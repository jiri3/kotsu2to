##### Visual Studio Code の 拡張機能（Extension） のサンプルを作る

有用な拡張機能かはおいといて、次のサンプルを作成してみました。  
どこかで紹介できればと思います。

- 現在時刻をドキュメントに挿入する
- ツリービューで選択中のフォルダを新しいウインドウで開く
- Picker で選択したファイルを markdown のリンク形式に変換してドキュメントに挿入する
- プロパティで設定したユーザー名とメールアドレスを git config で設定する
- CSV のプレビューを表示する
